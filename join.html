Rejoindre

Pour rejoindre le serveur simplement téléchargez ce launcher et connectez vous en utilisant votre EMAIL (pas le nom d'utilisateur comme écrit sur le launcher) et mot de passe minecraft ( les versions crack ne marchent pas) :

Launcher de Ninjin

[!] Assurez vous d'avoir Java 64bit sinon vous risquez les crashs [!]

Essayer ce lien de java 64bit si vous avez des problèmes : Java 64Bit

Si ça ne marche pas, vérifier bien que vous utilisez le Java situé sur Program Files/Java/jre1.8.0_ ... . Les java sur Program Files (x86) sont TOUJOURS du 32Bit.

Pour changer le chemin de java, cliquez sur options du launcher (Il faut être déconnecté pour voir le boutton) et changer pour votre chemin. Par exemple pour moi le chemin c'est : C:\Program Files\Java\jre1.8.0_201\bin\javaw.exe
